# showcase

Personal portfolio site — [csmntms.github.io/showcase](https://csmntms.github.io/showcase)

## Stack

Next.js 16 · TypeScript · Tailwind v4 · React Compiler · pnpm

Static export deployed to GitHub Pages via Actions.

## Local dev

```bash
cd apps/web
pnpm install
pnpm dev       # http://localhost:3000
```

## Deploy

Push to `main` → GitHub Actions builds and deploys automatically.
